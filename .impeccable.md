# Design Context

## Users

**目标用户**：自媒体新手，需要简单易用的引导
- 使用情境：个人创作公众号文章，追求高效完成内容创作
- 核心任务：快速撰写、排版、发布公众号文章
- 情感目标：自信、专注、愉悦的写作体验
- 关键需求：直观的操作引导、清晰的功能布局、舒适的视觉环境

## Brand Personality

**品牌个性**：专业、高效、简洁
- **专业**：提供可靠的写作工具，让内容创作更专业
- **高效**：优化操作流程，减少学习成本，提升创作效率
- **简洁**：去除冗余元素，聚焦核心功能，界面干净清爽

**情感目标**：让用户感到自信和掌控感，写作过程流畅无阻

## Aesthetic Direction

**视觉基调**：现代简约、专注写作、无干扰环境
- **色彩策略**：中性色为主，强调色点缀，支持浅色/深色双模式
- **排版风格**：清晰的视觉层次，舒适的阅读宽度和行高
- **空间设计**：充足的留白，合理的间距，呼吸感强的布局
- **动效设计**：微交互动效增强反馈，但不干扰专注

**参考风格**：
- 现代写作工具（如Notion、Typora）
- 专业内容创作平台
- 简约设计系统

**反参考**：
- 过度装饰的界面
- 复杂的仪表盘风格
- 过于花哨的动效

## Design Principles

### 1. 渐进式披露
- 常用功能一键可达
- 高级功能按需展示
- 新手引导清晰明了

### 2. 视觉层次清晰
- 重要信息突出显示
- 次要信息适度弱化
- 操作区域明确划分

### 3. 一致性原则
- 统一的交互模式
- 一致的设计语言
- 可预测的行为反馈

### 4. 专注写作环境
- 减少视觉干扰
- 舒适的阅读体验
- 沉浸式创作氛围

### 5. 响应式适配
- 不同屏幕尺寸的良好体验
- 移动端友好的操作
- 灵活的布局调整

## 技术约束

- **前端框架**：React 18 + TypeScript
- **样式系统**：Tailwind CSS + CSS变量
- **UI组件库**：shadcn/ui (Radix UI)
- **编辑器**：TipTap (StarterKit + 扩展)
- **图标库**：Lucide Icons
- **构建工具**：Vite
- **状态管理**：Zustand

## 特殊要求

- 支持浅色/深色模式切换
- 编辑器工具栏需要毛玻璃效果
- 编辑区域需要纸张质感
- 标签页需要文字提示（新手友好）
- 微交互动效增强用户体验
