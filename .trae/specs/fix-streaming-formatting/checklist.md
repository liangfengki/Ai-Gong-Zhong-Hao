# Checklist

- [ ] `markdownToHtml` 函数能正确将 markdown 标题、加粗、斜体、列表、引用、段落转为 HTML
- [ ] AI 流式生成的文章不再出现"每行两个字"的碎片化换行问题
- [ ] AI 输出的 `# 标题` 在编辑器中显示为 `<h1>` 标签
- [ ] AI 输出的 `**加粗**` 在编辑器中显示为 `<strong>` 标签
- [ ] AI 输出的列表、引用等 markdown 语法正确渲染
- [ ] 流结束后剩余 buffer 被正确 flush 到编辑器
- [ ] AI 改写（去AI味等）功能同样使用缓冲和 markdown 转换机制
- [ ] Prompt 中包含明确的 markdown 输出格式指令
- [ ] 一键排版模板能正确应用到转换后的 HTML 内容
- [ ] 现有编辑功能（手动编辑、撤销等）不受影响
